📖 关于「读我」

「读我」是一个纯前端 AI 代读系统。所有数据存在你的浏览器里，不经过任何服务器。

🔒 隐私说明

  • 你的 API Key 只存在浏览器本地（localStorage），不会上传到任何地方
  • LLM 调用直接从浏览器发到你选择的模型服务商
  • 报告数据存在浏览器 localStorage，清除浏览器数据会丢失
  • 建议定期使用「下载」功能备份报告

📋 报告结构

  • 🎯 这本书到底在解决什么问题？ — 核心问题 + 作者起心动念
  • 🔗 核心主张与论证链条 — 3-5 个主张，每个配书中案例
  • 💡 最颠覆直觉的 3 个观点 — 反常识观点 + 关键实验
  • 📖 如果只记住一个故事 — 全书最精彩故事的详细转述
  • 🤔 我可能会质疑的地方 — 逻辑漏洞与批判分析
  • 🛠️ 与我生活的接口 — 普通人能立刻用的行动建议

🛠️ 技术说明

纯静态单页应用，无后端、无数据库、无服务端渲染。LLM 调用通过浏览器 Fetch API 直连模型服务商的 OpenAI 兼容接口。文件解析使用 pdf.js（PDF）和 JSZip（EPUB）在浏览器内完成。
